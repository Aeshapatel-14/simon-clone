# Simon Game

An interactive memory-based Simon Game built using HTML, CSS, and JavaScript.

---

## Features

- Random color sequence generation
- User input validation
- Level progression
- Restart functionality
- Dynamic UI updates using DOM manipulation

---

## Tech Stack

- HTML5
- CSS3
- JavaScript
